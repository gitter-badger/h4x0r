About H4X0R

From the creators of The Legend of Adlez comes a new Team Ivan game, H4X0R. It's not a game about hacking, but what everybody thinks hacking is. For example, if you're a CSI writer, you probably think that hacking consists of multiple people mashing the same keyboard at the same time. And as we found out, that's fun. Case in point: GeekTyper. So we're making a 2-player game where you have to research on a fake Internet to learn how to "hack" and defeat your opponent.

Story

Here's the intro text that sums up the "story", if you can call it that, pretty succinctly: H4X0R is not a game about hacking. It's a game about what people think hacking is. Trust me; if hacking was anything like in movies, I would be too busy hacking the Pentagon to make this game. You, xXxGenericCoDPlayer42xXx, have heard word of these "hax". You must scour the Internet to absorb its vast knowledge. With enough patience and skill to defeat all that stop you, you too will become a leet haxor.

Copyright and stuff

The original game of H4X0R was made at DigiPen and was more of a proof-of-concept. Since the code for the old Java version is owned by them, I can't put it on here. However, I can start from scratch remaking the game with Phaser reusing the assets from the pre-alpha made in Java. The new version is made in HTML5, so if you can view this page, you can probably play it. It has a Python server for the multiplayer (Keep in mind that most of the new stuff doesn't exist yet, but soon...)
